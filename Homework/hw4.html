Answers To Questions

  • The box model is a way to describe the space arround HTML content. From the inner most layer the boxes are: content, padding, border, and Margin.
  • A block Level element is an elemen that starts on occupies the entire space available to it.
  • Many of the HTML tags we have learned so far are block-level elemnts: Form, p, ect.
  • Inline elements only take up the ammount of space required, and do not typically use a new line like block-level elements do
  • Some examples of inline elements are: img and input, which are both used in this page.
  • CSS can be included internally (in a style element in the head of the HTML page) or inline (within the ledt margin of an element).
  • Selectors are used to denote what HTML elements you want to style.
