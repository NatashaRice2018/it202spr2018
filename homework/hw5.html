# Answer
1 In order to uitilise Bootstrap in a webpage you can; 1. include bootstrap from a CDN or 2.download and host Bootstrap yourself.
2 CDN stands for content delivery network. This allows you to load Bootstrap code from remote servers.
3 Users will typically already have Bootstrap downloaded from a CDN prior to visiting your site. Allowing you to avoid loading it for them causing faster loading time when they visit your webpage.
4 In bootstrap a breakpoint allows for developers to change the scale of elements on the grid in responce to the viewing size changing.
5 There are 5 breakpoint values used in Bootstrap: extra-small=575.98px, small=576px, medium=768px, large=992px, and extra-large=1200px
6 The difference between fixed and full width is intuitive. Something with a fxed width cannot change size as the viewpoint changes, something with full width willl take up all space available.
7 Bootstrap Breaks the screen into 12 Columns.
8 In bootstrap the row class allows you to indicate that when possible the child elements should all be displaed in a singe row on the bootstrap grid system.
